What is Vue?

  • Yet another JavaScript framework!
  • Designed to be incrementally adoptable
  • Can be just your view (Vue!) layer, or a powerful single-page app with build process

Why is it awesome?

  • Companion libraries developed in tandem
  • The docs are great
  • Declarative vs imperative

Directives

  • HTML attributes with functionality offered by Vue
  • v-model: Two-way data binding
  • v-if, v-show: Conditional display
  • v-for: Easy looping through lists
  • v-bind or shorthand :
  • v-on or shorthand @

Vuex

  • Centralized store for data, logic, etc.
  • Flux architecture in Vue

  • You can use Redux too!

Actions

  • Asynchronous and most familiar from Redux
  • Contain business logic
  • Commit mutations from inside them
  • Good for when you need to do things in a certain order or hit a server
